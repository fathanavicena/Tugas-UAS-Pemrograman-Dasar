# UAS Pemrograman Dasar - S1 Teknik Komputer
**Mata Kuliah:** Pemrograman Dasar (TK091)
**Tema:** Manajemen Toko Elektronik

## Deskripsi Proyek
Proyek ini terdiri dari dua komponen utama:
1. **Sistem Mandiri**: Program manajemen stok menggunakan konsep OOP, List, dan Percabangan.
2. **Arsitektur Client-Server**: Implementasi REST API menggunakan Flask sebagai Server untuk penyimpanan data dan Streamlit sebagai Client (Antarmuka Pengguna).

## Cara Menjalankan Program
1. **Jalankan Server**: Buka terminal dan ketik `python server.py`.
2. **Jalankan Client**: Buka terminal baru dan ketik `streamlit run toko_elektronik.py`.
3. Akses aplikasi melalui browser di alamat yang tertera (biasanya localhost:8501).
